App Inventor 2 Project Merger Tool

Implemented by Arezu Esmaili

This tool combines two or more App Inventor 2 project source files to create one final project. Warning: This browser-based tool currently only works in Chrome and Firefox. In particular, it does not work in Safari or Internet Explorer.

Instructions:

Click the "Choose Files" button to select all project source files (.aia) you wish to merge. Keep in mind that a project can only have one "Screen1", so you will be asked to rename any additional "Screen1"s. Your projects will automatically merge once the files are chosen, and the final project will automatically be saved to your machine once you enter a name. The contents that are included in the final project from each file will be displayed on this page under "Contents:". You must refresh the page if you wish to merge again.
